signal()和kill()

signal()会接收要一个处理函数和对应的要处理的信号，返回值为调用signal()
前的该信号的处理函数指针。注册过处理函数后，进程再收到这个信号，系统就会回
调这个处理函数，并将收到的信号传递给这个处理函数。这样就可以将多个信号处理
函数注册为同一个函数，而在处理函数内部加以区分。
